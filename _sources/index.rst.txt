Documentazione del mio proj
=======================


Questo è una repo di test. In questo proj ho preso dei file a caso di python in modo da poter valutare la documentazione e vedere le potenzialità di sphinx.

Sto utilizzando un anaconda project per crearmi degli alias che mi creino facilmente la documentazione e il sito.

Utilizzando **anaconda-project prepare** viene eseguita una build del progetto (serve se aggiungi qualcosa).

I comandi implementati sono:

- **create_docs_project** per creare il progetto 
- **build_docs** serve per fare la build. Questo comando ti genera i file html
- **serve** è il comando che avvia un server sulla porta 8000, in modo da poter offrire la documentazione a tutti.
